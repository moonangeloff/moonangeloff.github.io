# moonangel.github.io
Функционал для описания продукта в Riot:

Авторизация и идентификация
Авторизация игроков через Riot Sign On (OAuth2) с перенаправлением на официальный сайт Riot и явным подтверждением доступа к данным.

Получение и использование идентификаторов аккаунта (puuid, ник и тег) после успешного входа.

Связка Riot‑аккаунта с Telegram‑пользователем и безопасное хранение только токенов и технических идентификаторов.

Автоматическое обновление access‑токенов по refresh‑токенам для сохранения доступа без повторной авторизации.

История матчей
Получение списка последних матчей авторизованного игрока с информацией о карте, режиме, времени матча и идентификаторе матча.

Загрузка детальной информации по выбранному матчу: результат (победа или поражение), счёт команд, сыгранный агент, KDA и основные индивидуальные показатели.

Отображение ключевых данных по каждому матчу в удобном текстовом формате внутри Telegram‑бота.

Ранг и соревновательная статистика
Получение актуального соревновательного ранга игрока и связанных с ним показателей прогресса.

Отображение текущего ранга, при возможности — численного рейтинга или прогресса в рамках эпизода/акта.

Формирование соревновательной статистики: количество ранговых матчей, побед и поражений, изменения ранга по последним играм.

Персональная статистика
Агрегация статистики по последним матчам: общий KDA, средний урон за раунд, количество побед и поражений, винрейт за выбранный период.

Подсчёт и показ статистики по агентам: число игр на каждом агенте, винрейт и средняя эффективность.

Статистика по картам и режимам: наиболее часто играемые карты, результаты и производительность игрока на разных картах и в разных режимах.

Ограничения и соответствие политикам
Доступ только к данным аккаунта самого авторизовавшегося игрока, строго по его согласию.

Отсутствие любых функций, дающих прямое игровое преимущество или напрямую управляющих игровым клиентом.

Соответствие общим политикам Riot: корректное описание продукта, отсутствие азартных и запрещённых механизмов, наличие бесплатного доступа к базовому функционалу.
